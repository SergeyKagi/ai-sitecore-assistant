# AI Sitecore Assistant

AI Sitecore Assistant is a pet project focused on designing and prototyping an AI-powered assistant for website content creation in Sitecore.

The goal is to reduce manual work for web content managers by converting marketing documents into structured page data that can later be mapped to Sitecore components.

## Target workflow

Marketing document  
→ AI parsing  
→ Structured JSON  
→ Component mapping  
→ Sitecore draft page  
→ QA checks  
→ Publish-ready page

## Initial scope

Version 1 focuses on:

- parsing marketing-provided content;
- identifying page structure;
- selecting suitable page components;
- generating structured JSON;
- generating SEO metadata;
- defining QA rules.

## Future scope

- Sitecore component mapping;
- browser automation;
- Sitecore API integration;
- automated draft page creation;
- automated QA validation.
